# Deno Codespace Setup

From the default codespace server:

* Install Deno
  * ```curl -fsSL https://deno.land/install.sh | sh```
* Install Deno Jupyter Server
  * ```deno jupyter --install```
* VSCode Extensions
  * Deno
  * Jupyter
